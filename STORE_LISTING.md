## Short Title
EGData Client

## Voice Title

## Short Description
Scan and upload Epic Games Store manifests for game preservation.

## Description
EGData Client is a free, open-source utility that helps preserve Epic Games Store data for research and archival purposes.

The application scans your local Epic Games installations and allows you to upload game manifest files to the EGData project. These manifests contain valuable metadata about game versions, updates, and file structures that help document the history of digital game distribution.

Whether you're a researcher, archivist, or simply want to contribute to game preservation efforts, EGData Client makes it easy to participate with just a few clicks.

The app runs entirely on your device and only uploads data when you choose to. No personal information is collected or transmitted.

## Features
- Automatically detects all installed Epic Games Store games
- Displays game information including name, version, and install size
- One-click upload of individual game manifests
- Batch upload all manifests at once
- Auto-sync feature for scheduled uploads
- Move games to different drives or folders
- Dark mode interface
- Lightweight and fast
